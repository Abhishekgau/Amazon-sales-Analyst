# Amazon Sales Data Analytics Project 📦

A Python-based data analysis project designed to analyze retail transaction records from an Amazon-style superstore dataset. This project uncovers sales performance trends, profitability metrics, and regional customer demand.

---

## 🛠️ Python Libraries Used
* **Pandas:** For cleaning transaction records, handling date-time parsing, and data manipulation.
* **NumPy:** For efficient mathematical and numerical metrics calculation.
* **Matplotlib & Seaborn:** For plotting statistical visualizations, monthly sales distributions, and category comparisons.

## 💡 Key Analysis & Insights
* **Sales Trends:** Analyzed revenue cycles to identify peak purchasing months and seasonal trends.
* **Category Profitability:** Evaluated which product categories and sub-categories drive the highest profit margins versus top sales volume.
* **Shipping & Logistics:** Studied the distribution of shipping modes and their impact on order delivery timelines.
* **Geographical Demand:** Mapped order density to identify high-performing states and cities.
